Noto is a global font collection for writing in all modern and ancient languages, with multiple weights and widths, in sans, serif, mono, and other styles. You can combine the Noto fonts with each other and with other fonts for harmonious, typographically correct communication in more than 1,000 languages and over 150 writing systems.

Noto Rashi Hebrew is modulated (“serif”) design for the Middle Eastern Hebrew script with a semi-cursive skeleton based on 15th-century Sephardic writing. It can be used for emphasis, complementing Noto Serif Hebrew. This type of design was used by early Hebrew typographers to set rabbinic commentary in the Talmud and Tanakh, while the primary text was set in a “square” Serif typeface. It has multiple weights, contains 92 glyphs, 3 OpenType features, and supports 91 characters.
