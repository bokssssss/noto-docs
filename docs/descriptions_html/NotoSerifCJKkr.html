Noto is a global font collection for writing in all modern and ancient languages, with multiple weights and widths, in sans, serif, mono, and other styles. You can combine the Noto fonts with each other and with other fonts for harmonious, typographically correct communication in more than 1,000 languages and over 150 writing systems.

Noto Serif CJK KR is a modulated (“serif”) design for the Korean language using Hangul and the Korean Hanja scripts. It covers all Hanja characters from the Basic Multilingual Plane of the Unicode Standard, and some additional ones. It also supports Hiragana, Katakana, Latin, Cyrillic and Greek. It contains 65,535 glyphs, 21 OpenType features, and supports 43,029 characters from 53 Unicode blocks.
