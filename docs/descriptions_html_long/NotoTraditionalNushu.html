Noto is a global font collection for writing in all modern and ancient languages, with multiple weights and widths, in sans, serif, mono, and other styles. You can combine the Noto fonts with each other and with other fonts for harmonious, typographically correct communication in more than 1,000 languages and over 150 writing systems.

Noto Traditional Nushu is an unmodulated (“sans serif”) design in multiple weights for the East Asian Nüshu script, with a calligraphic skeleton and a compact appearance. It is suitable for longer texts, especially those set in medium font sizes, and for headlines. It contains 872 glyphs, 2 OpenType features, and supports 472 characters from 3 Unicode blocks.
