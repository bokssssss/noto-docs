Noto is a global font collection for writing in all modern and ancient languages, with multiple weights and widths, in sans, serif, mono, and other styles. You can combine the Noto fonts with each other and with other fonts for harmonious, typographically correct communication in more than 1,000 languages and over 150 writing systems.

Noto Emoji is a monochrome (non-color) scalable OpenType format. It only contains about 25% of all encoded emoji symbols, and is intended for limited use in print publications, on monochrome displays, and in other scenarios where multiple colors cannot be reproduced. It contains 888 glyphs, and supports 887 characters from 11 Unicode blocks.
