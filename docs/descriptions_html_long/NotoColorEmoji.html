Noto is a global font collection for writing in all modern and ancient languages, with multiple weights and widths, in sans, serif, mono, and other styles. You can combine the Noto fonts with each other and with other fonts for harmonious, typographically correct communication in more than 1,000 languages and over 150 writing systems.

Noto Color Emoji is a multi-color font in the OpenType CBDT format. It supports all emoji symbols in Unicode 13 and beyond, but only works in Android, in Google Chrome and in apps that support the CBDT format. There is also a version of this font that works on Windows 10. It contains 3,113 glyphs, and supports 1,374 characters from 14 Unicode blocks.
