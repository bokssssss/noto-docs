Noto is a global font collection for writing in all modern and ancient languages, with multiple weights and widths, in sans, serif, mono, and other styles. You can combine the Noto fonts with each other and with other fonts for harmonious, typographically correct communication in more than 1,000 languages and over 150 writing systems.

Noto Sans Modi is a unmodulated (“sans serif”) design for texts in the Indic Modi script. It contains 209 glyphs, 7 OpenType features, and supports 96 characters from 2 Unicode blocks.
